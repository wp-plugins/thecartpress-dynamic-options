== Description ==
TheCartPress Dynamic Options is a plugin for TheCartPress eCommerce plugin.
It allows to add n-levels options (or variations) to the products, such as sizes, colors...

= TheCartPress Dynamic Options =

* Allows to manage Attributes and Attributes sets
* Supports Multi Lingual

= Unlimited features =

* TheCartPress-dynamicoptions is compatible with TheCartPress and has been developed by TheCartPress Team.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload TheCartpress Dynamic Options to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
